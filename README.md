# palm-data_veille
Outil de veille pour une entreprise d'archivage de données dans le domaine de la santé.
Elle couvre 4 type différents : Reglementaire, Concurrentiel, Commercial et Technique.
Le code est lançable sur tout pc sans adapter les path et peut importe l'OS (linux, macOS, windows)
Il ressort un fichier excel avec une page pour chaque type. 
Chaque type à respectivement : une synthèse globale, le titre du sujet, un résumé fait par IA et un score de pertinance.
Pour optimisé la veille, chaque information est récente, donc à été publiée quelques jours auparavant voir quelques heures seulement, afin de ne pas ressortir d'information déjà données dans les dernières veilles.
